# EasyMallssm
`EasyMallssm`使用SSM框架搭建电子商务平台，网页管理后台使用了X-Admin后台模板。含有登录/注册、购物车、订单、后台等模块。

# 使用
1. 使用git克隆或直接下载该项目
2. 使用`Eclipse`等导入该Java项目
3. 导入`EXAMPLE`文件夹中的`easymall.sql`数据库文件到`mysql`中
3. 由于编译使用的Java版本可能不一致，如果提示错误请根据您电脑的环境修改项目配置
4. 编译并运行
